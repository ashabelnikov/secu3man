ВЫХОДЫ:
IGN_Ox/IGN_OUTx - Выходы зажигания сильноточные и слаботочные (в зависимости от конструкции конкретного блока).

INJ_Ox/INJ_OUTx - Выходы впрыска сильноточные и слаботочные (в зависимости от конструкции конкретного блока).

ADD_Ox - Выходы дополнительные для разных целей.

ECF - Выход управления оборотами вентилятора.

IE - Выход управления клапаном ЭПХХ.

FE - Выход управления клапаном ЭМР.

STBL_O/ST_BLOCK - Выход управления реле стартера (блокировка стартера).

FPMP_O/FL_PUMP - Выход управления.

PWRR_O/PWRRELAY - Выход управления питанием блока МПСЗ/ЭСУД.

CEL_O - Выход для лампы 'Check Engine'.

EVAP_O - Выход управления клапаном продувки адсорбера.

O2SH_O - Выход управления подогревом ДК.

COND_O - Выход Управления муфтой кондиционера.

TACH_O - Выход сигнала для тахометра.

UNI_OUTx - Выходы универсальные программируемые.

SM_DIR - Выход импульсов шага для ШД РДВ ХХ/ВЗ.

SM_STP - Выход сигнала направления ШД РДВ ХХ/ВЗ.

GD_DIR - Выход импульсов шага для ШД дозатора газа.

GD_STP - Выход сигнала направления для ШД дозатора газа.

IAC_PWM - Выход ШИМ для РДВ ХХ (клапан моментного типа).

GD_PWM - Выход ШИМ для дозатора газа (дозатор управляемый ШИМ).

GASVAL_O - Выход управления газовым клапаном.

GRHEAT - Выход управления подогревом газового редуктора.

VTACHOM - Выход универсальный настраиваемый для тахометра.

HALL_OUT - Выход эмуляции сигнала с ДХ трамблера.

INTK_HEAT - Выход управления подогревом впускного коллектора.

STROBE - Выход для лампы стробоскопа (светодиод).

PWM1/PWM2 - Выходы универсальные ШИМ с управлением по таблицам от (нагрузка, обороты).

FL_CONS - Выход частоты сигнала мгновенного расхода топлива.

KSP_CS - Выход управления микросхемой TPIC8101/HIP9011.

CAN_CS - Выход управления микросхемой CAN-шины MCP2515.

ETC_PWM1 - Выход ШИМ для движения электронной заслонки в сторону открытия.
 
ETC_PWM2 - Выход ШИМ для движения электронной заслонки в сторону закрытия.


ВХОДЫ:
BL - Вход/Выход для различных целей. Вход запуска загрузчика. Слабо защищенный, низкоточный.

DE - Вход/Выход для различных целей. Вход сброса EEPROM. Слабозащищенный, низкоточный.

CKPS - Вход дифференциального типа для подключения ДПКВ/ДУИ.

PS - Вход для датчика фаз на эффекте Холла и других целей.

REF_S - Вход для датчика фаз индукционного (VR) типа и других целей.

GAS_V - Вход состояния газового клапана.

IGN/IGN_I - Вход состояния замка зажигания.

EPAS_I - Вход запроса на включение ЭУР.

COND_I - Вход запроса на включение муфты кондиционера.

MAP_S/MAP_I - Вход ДАД.

ADD_Ix - Входы дополнительные для разных целей.

LAMBDA - Вход датчика кислорода 1.

LAMBDA2 - Вход датчика кислорода 2.

AIR_TEMP - Вход датчика температуры воздуха на впуске.

SPD_SENS - Вход датчика скорости.

MAP2 - Вход ДАД2 (датчик атмосферного давления).

GPS - Вход ДДГ (датчик давления газа ГБО).

TEMP2 - Вход датчика температуры газа.

MAF - Вход ДМРВ.

MAPSEL0 - Вход для реализации переключения между 4 наборами таблиц.

BC_INPUT - Вход запуска режима блинк кодов (check Engine).

IGNTIM - Вход ручной коррекции УОЗ.

INJPWC_I - Вход ручной коррекции длительности впрыска.

AUTO_I - Вход изменения УОЗ при переключении передач.

INPUTx - Вход для использования в универсальных программируемых выходах.

FTLS_I - Вход датчика уровня топлива в баке.

EGTS_I - Вход датчика температуры выхлопных газов.

OPS_I - Вход датчика давления масла.

FTS_I - Вход датчика температуры топлива.

REFPRS_I - Вход датчика давления хладагента кондиционера.

ALTRN_I - Вход состояния возбуждения генератора.

FPS - Вход датчика давления топлива.

TPS2 - Вход датчика положения дроссельной заслонки 2 (электронный дроссель). На нём напряжение падает по мере открытия заслонки.

APPS1 - Вход датчика положения педали акселератора 1 (электронная педаль газа). На нём напряжение в 2 раза выше напряжения на APPS2.

APPS2 - Вход датчика положения педали акселератора 2 (электронная педаль газа). На нём напряжение в 2 раза ниже напряжения на APPS1.

OTS_I - Вход датчика температуры масла.
